# README - en

## What's this?

My Dotfiles.

## Purpose

Manage my all configs, fonts and wallpapers in this repository.

## Requirements

- git
- GNU make

## Usage

```sh
git clone git@github.com:yuni-hutsuka/yunifiles.git ~/.yunifiles
cd .yunifiles
# if your shell says "Permission denied.", then
# chmod +x boot.sh
./boot.sh
```
